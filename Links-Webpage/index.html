My Favorite Operating Systems

  • Solus Linux is an up and coming distro built from scratch using a brand spanking new Desktop Environment--Budgie!
  • Ubuntu-- the best linux distro for beginners! It's also really good for servers (althought Ubuntu Server doesn't come with a desktop environment)
  • Fedora! A linux distro runing pure GNOME. No additions or tweaks like Canonical with Ubuntu.
    Fedora is targeted at developers
  • Manjaro Linux! The stability of the legendary Arch Linux, but without the horrendously difficult installation
    procedure that Arch is known for
  • Antergos Linux! Arch, but without the horrible install procedure, and GNOME. 'Nuff said.

My Favorite Desktop Environments

  • Budgie! Budgie is a DE that looks similar to Windows, but is better in a number of ways.
    It can be found running on a number of distros, including Ubuntu Budgie, Solus (the flagship distro using Budgie), and Manjaro Linux.
  • GNOME! Some regard it as a somewhat "heavier" environment (meaning it uses up more resources compared to other DEs. Compared to Windows, its still a lightweight. GNOME's flagship distro is Fedora, but it can be found running on literally almost anything. It takes some getting used to but once one gets used to the different window management, GNOME is rock-solid.
  • KDE! A DE for more advanced users, as it exposes almost every switch you could possibly want to toggle in the GUI without having to do terminal wizardry. KDE's flagship distro is KDE Neon, and it's developed by the same team who makes the DE itself
  • Cinnamon! A desktop environment that has it's claim to fame in Linux Mint-- An Ubuntu-based Linux distribution that's great for beginners!
  • Unity! Unity was used by Canonical for Ubuntu in version prior to 18.04. in Ubuntu 18.04 Unity was abandoned in favor of GNOME (but with heavy customizations)

The Worst Versions of Windows

  • Windows Versions prior to 3.1-- These versions of Windows had very little in the way of multitasking. Wanted to overlap windows? You can't. Windows can only be tiled. After Windows 3.1, overlapped windows were enabled as a feature.
  • Windows ME-- Windows Millenium Edition is Windows 98, with all of the bad parts and a pastel color scheme. It was b a d
  • Windows Vista-- Windows 7's disgraced cousin. 7 is everything Vista tried to be. but alas, Vista was a touch too ambitious. The system requirements were high, and crashes were rampant!
  • Windows 8-- Windows 8 was also a disgraced cousin. Windows 10 is everything Windows 8 was supposed to be: A melding between a tablet-style UI and a desktop UI that actually works
  • Windows 2000-- Windows 2000 actually was not that bad. It was the business version of Windows ME, but it was built on a different kernel (the New Technology kernel, or NT kernel). Prior versions of Windows had essentially been a layer over top of MS-DOS (MicroSoft-Disk Operating System), but Windows 2000 was the first time Windows could become independent of DOS. Tech pioneered with Windows 2000 first appeared in consumer releases of Windows in Windows XP

Fun Stuff

  • Star Wars, but text! If you go to any terminal emulator (excepting Windows Command Prompt and PowerShell. PuTTY can be used to get this easter egg on Windows)
  • A picture of Apple Park in the Finder about window: Press and hold Option, then use the mouse and click "About the Finder". A picture of Apple Park will pop up instead of the usual About window.
  • A Poem in TextEdit! If you look at the current icon for Text Edit in the macOS, it will have a poem in it! It reads as follows: Dear Kate, Here’s to the crazy ones. The misfits. The rebels. The troublemakers. The round pegs in the square holes — the ones who see things differently. They’re not fond of rules and they have no respect for the status quo. You can praise them, disagree with them, quote them, disbelieve them, glorify, or vilify them. About the only thing that you can’t do is ignore them. Because they change things. Take care,John Appleseed
  • Cool slow-motion effects when interacting with a window! If you press and hold Shift, then interact with the Maximize or Minimize button, the action will be in slow motion!
  • When there is an in progress app download on macOS, the "date modified" field will read January 24, 1984. This date was the debut of the Macintosh!

Google Easter Eggs

  • When you type in "askew", the Google window will actually rotate to the side slightly.
  • Typing in Atari Breakout in a Google image search, and play away!
  • If you search for "the answer to life, the universe and everything", the Google calculator pops up with 42 in the screen.Its a reference to "Hitchhiker's Guide to the Galaxy"
  • Bletchly Park-- if you search for Bletchly Park on Google, the Knowledge Panel will have the name scrambled. It will then decode.
  • Sonic the hedgehog in Google search! If you search for Sonic the Hedgehog, there will be an interactable Classic Sonic in the Knowledge Panel!
